# 🚀 Earvollt - Sistema de Monetización, Tracking y Pagos

Earvollt es una solución backend robusta construida en **Node.js** diseñada para gestionar la autenticación de usuarios, realizar el seguimiento (tracking) de actividades y procesar pagos y recompensas de forma segura.

---

## 🛠️ Características del Sistema
*   **Autenticación Segura:** Control de acceso y gestión de usuarios.
*   **Tracking de Conversiones:** Monitoreo en tiempo real de las acciones de los usuarios.
*   **Pasarela de Pagos integrada:** Endpoint seguro para recibir postbacks de **Lootably** y procesar recompensas/monetización de forma automática.

## 📦 Arquitectura del Proyecto
*   `/` (Raíz): Contiene el servidor principal (`server.js`) configurado con Node.js y Express.
*   *(Opcional)* `/public` o `index.html`: Interfaz visual para que el usuario interactúe con el simulador/sistema.
